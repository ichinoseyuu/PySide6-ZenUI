## initialization

Recommend using uv tools to manage projects

```powershell
uv init
uv sync
uv pip install -e .
```

## build

```powershell
uv build
```

## install

```powershell
uv add ./dist/zenui-0.1.0-py3-none-any.whl #or
uv add ./dist/zenui-0.1.0.tar.gz
```
